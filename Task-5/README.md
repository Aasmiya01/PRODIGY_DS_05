# Task 5

## Traffic Accident Data Analysis

### Project Overview

This project focuses on analyzing traffic accident data to identify patterns related to road conditions, weather, and time of day.

The analysis helps in understanding accident hotspots and the major contributing factors responsible for road accidents.

Data visualization techniques are used to represent accident trends and insights effectively.

---

## Objective

The main objectives of this project are to:

* Analyze traffic accident data
* Identify accident patterns based on:

  * Road conditions
  * Weather conditions
  * Time of day
* Detect accident-prone hotspots
* Visualize contributing factors through charts and graphs

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## Project Workflow

### 1. Data Collection

Traffic accident datasets are collected from available sources.

### 2. Data Cleaning

The dataset is preprocessed by:

* Removing missing values
* Handling duplicate records
* Formatting date and time columns

### 3. Data Analysis

Analysis is performed to identify patterns based on:

* Weather conditions
* Road surface conditions
* Day/Night accident frequency
* Peak accident hours

### 4. Visualization

Data is visualized using:

* Bar Charts
* Heatmaps
* Pie Charts
* Accident hotspot maps
